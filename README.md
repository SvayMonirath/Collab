# SAMAKI
Samaki is a collaborative platform designed to enhance team productivity and communication.

## Features
- Meeting Transcription: Real-time transcription of meetings to capture important discussions.
- AI Summarization: Generate concise summaries of meetings using advanced AI algorithms.
- Automatic Suggestion: Review and Approve action recommended by AI base on meeting discussions.
- Team Collaboration: Create teams, manage members, and facilitate seamless collaboration.
